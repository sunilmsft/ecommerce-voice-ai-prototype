Enter a PAT with Work Items (Read) scope. Stored in session only. How do I get a PAT?
📋 Create a Personal Access Token (PAT) — takes ~30 seconds
  1. Open MicrosoftIT → Personal Access Tokens
  2. Click + New Token
  3. Name it anything (e.g. "Handshake Prototype")
  4. Set Organization = MicrosoftIT
  5. Set Expiration = 30 days (or custom)
  6. Under Scopes, select Work Items → Read only
  7. Click Create → copy the token → paste it above
⚠️ Security note: Your PAT is only stored in this browser tab's session. It is never saved to disk, sent to any server other than dev.azure.com/MicrosoftIT, or visible to other users. Closing the tab clears it.
